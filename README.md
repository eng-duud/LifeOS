# Life OS - نظام تشغيل الحياة الشخصي

تطبيق ويب شامل لإدارة الإنتاجية الشخصية يجمع بين إدارة المهام والأهداف والعادات والمشاريع والكتب والخطط والتقويم والإحصائيات.

## 🎯 الميزات الرئيسية

### 📋 إدارة المهام
- إنشاء وتعديل وحذف المهام
- تصنيف حسب الأولوية (عالية، متوسطة، منخفضة)
- تحديد حالات المهام (قيد الانتظار، قيد الإنجاز، مكتملة)
- تحديد تواريخ الاستحقاق
- فلترة وترتيب متقدم

### 🎯 إدارة الأهداف
- تحديد أهداف قصيرة وطويلة الأجل
- تتبع التقدم بصريًا
- ربط الأهداف بالمهام والمشاريع
- عرض نسبة الإنجاز

### 🚀 إدارة المشاريع
- إنشاء وإدارة المشاريع
- تتبع حالة المشروع (جاري، مكتمل، معلق)
- إدارة مهام المشروع الفرعية
- عرض نسبة إنجاز المشروع

### 🔄 تتبع العادات
- إنشاء عادات يومية وأسبوعية
- تسجيل إكمال العادات
- عرض السلسلة الحالية والأطول
- تقويم بصري للسلسلة

### 📚 مكتبة الكتب
- تتبع الكتب المقروءة والقيد القراءة
- إضافة تقييمات وملاحظات
- عرض نسبة القراءة
- فلترة حسب حالة القراءة

### 📅 الخطط والتقويم
- خطط يومية وأسبوعية وشهرية
- عرض تقويمي شامل
- عرض المهام والعادات والخطط على التقويم
- تتبع الأحداث المهمة

### 📊 الإحصائيات والتحليلات
- رسوم بيانية للمهام والأهداف والمشاريع
- تحليل العادات
- تقارير الإنتاجية
- سجل النشاط

### 🤖 المساعد الذكي
- استشارات وتوصيات ذكية
- نصائح لتحسين الإنتاجية
- إجابات على الأسئلة المتعلقة بالإدارة

### 🎨 الإعدادات والتخصيص
- تبديل المظهر الداكن/الفاتح
- إدارة الإشعارات
- إعدادات اللغة
- معلومات الحساب

## 🛠️ المتطلبات التقنية

- Node.js 18+
- pnpm 10+
- MySQL/TiDB قاعدة بيانات
- متصفح حديث يدعم PWA

## 📦 التثبيت والتشغيل

### التثبيت المحلي

```bash
# استنساخ المشروع
git clone <repository-url>
cd life-os-prod

# تثبيت الاعتماديات
pnpm install

# إعداد متغيرات البيئة
cp .env.example .env
# عدّل .env بمعلومات قاعدة البيانات الخاصة بك

# تطبيق الهجرات
pnpm drizzle-kit generate
pnpm drizzle-kit migrate

# تشغيل خادم التطوير
pnpm dev
```

الآن يمكنك الوصول للتطبيق على `http://localhost:3000`

### البناء للإنتاج

```bash
# بناء التطبيق
pnpm build

# تشغيل الخادم الإنتاجي
pnpm start
```

## 📱 ميزات PWA

التطبيق يدعم تثبيته كتطبيق ويب تقدمي (PWA):

- **التثبيت على الشاشة الرئيسية** - ثبّت التطبيق مباشرة على هاتفك
- **العمل بلا إنترنت** - استخدم التطبيق حتى بدون اتصال إنترنت
- **الإشعارات** - احصل على تنبيهات للمهام والعادات
- **المزامنة الخلفية** - مزامنة البيانات تلقائياً عند الاتصال

## 🔐 الأمان

- **المصادقة الآمنة** - تسجيل دخول آمن عبر OAuth
- **تشفير البيانات** - جميع البيانات محمية بالتشفير
- **HTTPS فقط** - جميع الاتصالات آمنة
- **CSRF Protection** - حماية من هجمات CSRF
- **SQL Injection Prevention** - استخدام Drizzle ORM

## 📚 API Documentation

### المهام (Tasks)

```typescript
// الحصول على قائمة المهام
trpc.tasks.list.useQuery()

// إنشاء مهمة جديدة
trpc.tasks.create.useMutation({
  title: string,
  description?: string,
  priority?: 'low' | 'medium' | 'high',
  status?: 'pending' | 'in_progress' | 'completed',
  dueDate?: Date
})

// تحديث مهمة
trpc.tasks.update.useMutation({
  id: number,
  ...fields
})

// حذف مهمة
trpc.tasks.delete.useMutation({
  id: number
})
```

### الأهداف (Goals)

```typescript
// الحصول على قائمة الأهداف
trpc.goals.list.useQuery()

// إنشاء هدف جديد
trpc.goals.create.useMutation({
  title: string,
  description?: string,
  targetDate?: Date,
  progress?: number
})

// تحديث هدف
trpc.goals.update.useMutation({
  id: number,
  ...fields
})

// حذف هدف
trpc.goals.delete.useMutation({
  id: number
})
```

### العادات (Habits)

```typescript
// الحصول على قائمة العادات
trpc.habits.list.useQuery()

// إنشاء عادة جديدة
trpc.habits.create.useMutation({
  title: string,
  description?: string,
  frequency: 'daily' | 'weekly' | 'monthly'
})

// تسجيل إكمال العادة
trpc.habits.recordCompletion.useMutation({
  habitId: number,
  completedAt: Date
})
```
